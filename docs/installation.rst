============
Installation
============


Stable release
--------------

To install inspyred, run this command in your terminal:

.. code-block:: console

    $ pip install inspyred

This is the preferred method to install inspyred, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for inspyred can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/aarongarrett/inspyred

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/aarongarrett/inspyred/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ pip install .


.. _Github repo: https://github.com/aarongarrett/inspyred
.. _tarball: https://github.com/aarongarrett/inspyred/tarball/master
